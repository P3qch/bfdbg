# bfdbg
A very simple brainfuck visualizer with a nice TUI

## Why 
cuz i was bored lol

## Usage
compile it and use

```bash
bfdbg FILENAME_GOES_HERE
```

Keys:
- s - step
- Right/Left - Scroll the memory tape
- Up/Down - Scroll the output
